# ETL Pipeline for Music Data Analytics

This project implements an ETL pipeline to ingest Spotify artist/track data using AWS Lambda and Spotipy.

## Features:
- Automated Snowflake loading with Snowpipe and event-based S3 triggers.
- Designed star-schema models for analytics on listening patterns and trends.
- Added incremental ingestion to reduce compute cost and improve efficiency.

## Technologies Used:
- Python
- AWS Lambda
- Amazon S3
- Snowflake
- Spotipy
